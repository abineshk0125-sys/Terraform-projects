# 🚀 Terraform Remote Backend on AWS

## 📌 Project Overview

This project demonstrates how to configure a **remote backend in Terraform** using AWS services.

## 🧰 Technologies

* Terraform
* AWS S3 (state storage)
* AWS DynamoDB (state locking)
* AWS EC2 (infrastructure)

## ⚙️ What I built

* S3 bucket to store Terraform state remotely
* DynamoDB table for state locking
* EC2 instance deployment using Terraform

## 🧠 Key Learnings

* Terraform backend initializes before resources
* Importance of state locking
* Real-world Infrastructure as Code workflow

## 🚀 How to run

```bash
terraform init
terraform plan
terraform apply
```

## 📸 Output

* EC2 instance created
* State stored in S3
* Locking handled via DynamoDB
